
# Logs

This directory contains log files generated by Slurm jobs.

## OpenMP logs

- Files named `output32_tX.log` are OpenMP runs where `X` is the number of threads used.
- The matching error logs are named `error32.log` (or similarly named error files when present).

## CUDA logs

- Files named `outputcudalcs*.log` are CUDA runs.
- The matching error logs are named `errorcudalcs*.log`.

## Profiling logs

- Files named `outputcudalcs_prof.log` are CUDA profiling runs.
- Profiling output also includes `*.nsys-rep` and `*.sqlite` files.
